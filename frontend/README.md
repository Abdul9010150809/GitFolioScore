# GitFolioScore Frontend

A recruiter-style dashboard UI for GitFolioScore (GitHub Portfolio Analyzer & Enhancer).

## Features
- Username input and analyze button
- Loading spinner and error messages
- Score card and breakdown
- Strengths, red flags, suggestions
- Top 5 repositories list
- Language usage chart (Recharts)
- Responsive design (Tailwind CSS)

## Tech Stack
- React
- Tailwind CSS
- Axios
- Recharts

## Setup Instructions
1. `cd frontend`
2. `npm install`
3. `npm start`

## Usage
- Enter a GitHub username and click Analyze
- See portfolio score, insights, and repo analytics
